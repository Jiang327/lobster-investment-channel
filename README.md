# 小龙虾投资频道 🦞📈

专业的投资交易分析网页，采用金融交易风格设计，提供实时行情、市场分析和交易功能。

## 功能特点

- **实时行情**：沪深300、比特币、苹果、特斯拉等主要资产价格
- **投资组合**：个人持仓展示与盈亏分析
- **财经新闻**：最新市场动态与政策解读
- **快速交易**：模拟买入/卖出功能
- **响应式设计**：适配手机、平板和电脑

## 技术栈

- HTML5 / CSS3 / JavaScript (原生)
- Font Awesome 图标库
- 纯前端实现，无需后端

## 部署到 GitHub Pages

此网页已优化为适合 GitHub Pages 部署。访问地址：
`https://[你的用户名].github.io/[仓库名]/lobster-investment-channel.html`

## 本地运行

```bash
# 使用 Python 简单 HTTP 服务器
python3 -m http.server 8000

# 然后在浏览器访问
open http://localhost:8000/lobster-investment-channel.html
```

## 文件结构

```
├── lobster-investment-channel.html  # 主网页文件
```

## 自定义修改

1. 修改股票数据：编辑 HTML 文件中的 JavaScript 部分
2. 调整颜色主题：修改 CSS 中的颜色变量
3. 更新新闻内容：直接编辑 HTML 中的新闻部分
